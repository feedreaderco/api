DEPRECATED API

This API is deprecated. Please use the new API instead. Mail me if you need help.

Tokens

POST /:user/tokens/json

Authenticates a session. Accepts password as a parameter and returns an authentication token.

curl https://feedreader.co/arpith/tokens/json -F password=notreallymypassword

{ "success": true, "token": "fb8ac39f556118deef404bb7d0c0192f7d483759c348694d" }

DELETE /:user/tokens/json

Revokes a session. Requires authentication.

curl -X DELETE https://feedreader.co/arpith/tokens/json -u fb8ac39f556118deef404bb7d0c0192f7d483759c348694d:

{ "success": true }

Feeds

POST /:user/feeds/json

Add feed url (parameter names: xmlurl, title, link) or feeds from an opml file (parameter name: opml). Requires authentication.

curl https://feedreader.co/arpith/feeds/json -F opml=@subscriptions.xml -u fb8ac39f556118deef404bb7d0c0192f7d483759c348694d:

{ "success": true }

GET /:user/feeds/json

Get all subscribed feeds. Returns a list of feed objects and articles not marked read.

curl https://feedreader.co/arpith/feeds/json

{ "success": true, "feeds": [ { "title": "American Elf", "htmlUrl": "http://www.americanelf.com", "type": "rss", "lastModified": "undefined", "etag": "undefined", "key": "http://feeds.feedburner.com/AmericanElfRssFeed" } ], "articles": [] }

GET /feeds/:xmlurl/json

Checks url for new articles and returns feed object with sorted list of article hashes.

curl https://feedreader.co/arpith/feeds/http://feeds.feedburner.com/AmericanElfRssFeed/json -u fb8ac39f556118deef404bb7d0c0192f7d483759c348694d:

{ "title": "American Elf", "htmlUrl": "http://www.americanelf.com", "type": "rss", "lastModified": "undefined", "etag": "undefined", "success": true, "articles": [ "54eac3bad961867dfc67b659a027f0d4", "97bee92629ecf71804dc4505080c68da", "b3763bd521a5ef1c81443e1448cdc051" ] }

Labels

POST /:user/labels/:label/json

Label an article. Accepts hash (required) and score (optional) as parameters. If score is not present, Date.now() is used to order the article within the label. Requires authentication.

curl https://feedreader.co/arpith/labels/read/json -F hash=cc5e5dcb16010782336e0e1a7e98395e -u fb8ac39f556118deef404bb7d0c0192f7d483759c348694d:

{ "success": true }

GET /:user/labels/:label/json

Get articles labelled with label.

curl https://feedreader.co/arpith/labels/read/json -u fb8ac39f556118deef404bb7d0c0192f7d483759c348694d:

{ "success": true, "articles": [ "cc5e5dcb16010782336e0e1a7e98395e" ] }

GET /:user/labels/json

Get all labels used.

curl https://feedreader.co/arpith/labels/json -u fb8ac39f556118deef404bb7d0c0192f7d483759c348694d:

{ "success": true, "labels": [ "read" ] }

Folders

POST /:user/folders/:folder/json

Add a feed to a folder. Accepts xmlurl as parameter. Requires authentication.

curl https://feedreader.co/arpith/folders/Kochalka/json -F xmlurl=http://feeds.feedburner.com/AmericanElfRssFeed -u fb8ac39f556118deef404bb7d0c0192f7d483759c348694d:

{ "success": true }

DELETE /:user/folders/:folder/json

Delete a feed from a folder. Accepts xmlurl as parameter and deletes the folder if it is now empty. Requires authentication.

curl https://feedreader.co/arpith/folders/Kochalka/json -X DELETE -F xmlurl=http://feeds.feedburner.com/AmericanElfRssFeed -u fb8ac39f556118deef404bb7d0c0192f7d483759c348694d:

{ "success": true }

GET /:user/folders/:folder/json

Get feeds in folder.

curl https://feedreader.co/arpith/folders/Kochalka/json

{ "success": true, "feeds": [ "http://feeds.feedburner.com/AmericanElfRssFeed" ] }

GET /:user/folders/json

Get all folders used. If xmlurl is passed as a parameter, returns folders the feed belongs to as well as all folders used.

curl https://feedreader.co/arpith/folders/json?xmlurl=http://xkcd.com/atom.xml

{ "success": true, "folders": [ "Other" ], "allFolders": [ "Other" ] }

Articles

POST /articles/json

Generates a hash for identification. Accepts pubdate, link and xmlurl as parameters, and returns a hex digest of the sha1 hash.

curl https://feedreader.co/articles/json -F pubdate=2012-10-15T05:50:00.000Z -F link=http://feedproxy.google.com/~r/AmericanElfRssFeed/~3/RyYvUlNe5jw/americanelf.php -F xmlurl=http://feeds.feedburner.com/AmericanElfRssFeed -u fb8ac39f556118deef404bb7d0c0192f7d483759c348694d:

Note: This is currently generating a different hash and needs to be fixed.

{ "success": true, "hash": "cf7a11a8d058e60cacd3fd3d26bcbb3f" }
